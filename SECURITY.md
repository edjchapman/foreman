# Security Policy

## Reporting a vulnerability

Please report security issues **privately** via GitHub's
[private vulnerability reporting](https://github.com/edjchapman/foreman/security/advisories/new)
— do not open a public issue for anything security-sensitive.

You can expect an initial response within a few days. This is a portfolio project
with no production deployment or SLA, but reports are welcome and will be addressed.

## Supported versions

The latest release on `main` is the only supported version.
